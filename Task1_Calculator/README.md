## 📌 Project Overview

CalcGlass is a modern and responsive calculator application developed as part of the **Oasis Infobyte Internship Program (OIBSIP) - Web Development and Designing Domain, Level 2 Task 1**.

The calculator provides a simple and interactive experience with a stylish glassmorphism user interface, animated background effects, calculation history, and keyboard support.

---

## ✨ Features

- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)

- Modern glassmorphism UI design

- Animated gradient background

- Responsive design for all screen sizes

- Keyboard input support

- Calculation history storage

- Clear (AC) function

- Percentage calculation

- Positive/Negative value conversion

- Error handling for invalid calculations

- Interactive button animations

---
