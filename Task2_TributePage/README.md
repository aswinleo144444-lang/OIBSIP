# 🏏 Virat Kohli Tribute Page

---

## 📌 Project Overview

This project is a tribute webpage dedicated to **Virat Kohli - The Chase Master**, created as part of the **Oasis Infobyte Internship Program (OIBSIP) - Web Development and Designing Domain, Level 2 Task 2**.

The webpage highlights Virat Kohli's cricket journey, achievements, records, career milestones, and contributions to Indian cricket through a modern and visually engaging design.

---

## ✨ Features

- 🏏 Hero section with Virat Kohli introduction

- 📖 About section showcasing personal and cricket details

- 📅 Career timeline displaying important milestones

- 🏆 Honours and achievements section

- 📊 World records showcase

- 💬 Inspirational quote section

- 🎨 Modern dark-themed UI design

- 🔥 Smooth animations and transitions

- 📱 Fully responsive layout for all devices

---

- JavaScript
