## 📌 Project Overview

NeonTask is a modern and interactive **To-Do List Application** developed as part of the **Oasis Infobyte Internship Program (OIBSIP) - Web Development and Designing Domain, Level 2 Task 3**.

The application helps users manage daily tasks efficiently by allowing them to add, edit, complete, and delete tasks with a visually attractive neon-themed user interface.

---

## ✨ Features

- ➕ Add new tasks

- ✏️ Edit existing tasks

- ✅ Mark tasks as completed

- 🗑️ Delete tasks

- 📋 Separate sections for:
  - Pending Tasks
  - Completed Tasks

- 📊 Task statistics dashboard:
  - Total tasks
  - Pending tasks
  - Completed tasks

- 📈 Dynamic progress tracking

- ⌨️ Enter key support for adding tasks

- 🪟 Edit task popup modal

- 🎨 Modern neon dark theme design

- 📱 Responsive layout for different screen sizes
