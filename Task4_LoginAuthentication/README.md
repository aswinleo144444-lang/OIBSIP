## 📌 Project Overview

CipherGate is a secure and interactive **Login Authentication System** developed as part of the **Oasis Infobyte Internship Program (OIBSIP) - Web Development and Designing Domain, Level 2 Task 4**.

The project allows users to register, login, and access a secured dashboard through a simple authentication workflow with form validation and a modern cybersecurity-inspired interface.

---

## ✨ Features

- 📝 User Registration System

- 🔑 User Login Authentication

- ✅ Email validation

- 🔒 Password validation

- 👁️ Show/Hide password feature

- 💪 Password strength indicator

- ⚠️ Error handling and validation messages

- 🔐 Secure dashboard access after login

- 👤 User profile display

- 🕒 Login time and date tracking

- 🚪 Logout functionality

- 🎨 Modern dark cybersecurity-themed UI design

- 📱 Responsive design for different devices

---

## 🚀 How to Run the Project

1. Clone or download this repository.

2. Open the project folder.

3. Open `index.html` in any web browser.

4. Register a new account.

5. Login using the registered credentials.

6. Access the secured dashboard.
